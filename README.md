# Программа по поиску пропущенных параграфов и проблем нумерации
Виды параграфов:
```1. 1) 1.1 (1)``` числа

```a. a) (a)``` буквы

```I. I)``` римские цифры

(Ещё табуляцию не забываем и пробелы \t\r)
